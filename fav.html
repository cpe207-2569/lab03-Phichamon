Web Application

Progressive Web App (PWA)

CTF (Capture The Flag)

เกมการแข่งขันด้านความปลอดภัยทางไซเบอร์ โดยผู้เข้าแข่งขันจะต้องแฮ็ก ค้นหา หรือถอดรหัสเพื่อดึง Flag ที่ซ่อนอยู่ในระบบหรือโจทย์ที่จำลองขึ้นมา เพื่อรับคะแนน
